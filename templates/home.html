Terms of Use


Student Grade is used to store the student Internal marks like Mid marks and Lab marks. It must be accessible to the limited Faculty memebers of different depatments of a college.


The login Id and password will be given by the admin and he will have access to add/delete an user


Student:


Student tab is used to add ,delete and view students which are stored. In the student tab some fields are optional but you must fill all the fields during adding and deleting an student.In other cases those are optional


Marks:


Marks tab is used to store the marks gained by the student in a subject.Make sure that you have stored that subject previously.It also have Regd.No as optional field but it is compulsory when we modify marks as modify marks is used for modification of marks of a particular student.


Subject:


Subject tab is used to add or delete subjects or view subjects.It has one optional field(subject name) but it is compulsory during adding and deleting subjects.


Please make sure that you have properly understand usage of Student Grade . If there is an issue please contact us.
